Detailed Description

The abstracted window class that is used for the window facade. This class holds all the Glfw window information inside of it and it is an abstract class of Window. It is this way as it allows us to swap the potential window system if we ever wanted to add DirectX or Vulkan. The class handles all aspects of the window, anything that would require it such as size, inputs, camera, time, projections and window updates.
